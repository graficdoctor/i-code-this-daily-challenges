The Accordion

The accordion is a graphical control element comprising a vertically stacked list of items, such as labels or thumbnails. Each item can be "expanded" or "collapsed" to reveal the content associated with that item.

When and how it should be used?

Show/Hide operation

An accordion always contains the category title, an expanded and a collapsed state, an icon indicating expansion, and the spacing between them.

Tabbed interface

A list of items where exactly one item is expanded into a panel
